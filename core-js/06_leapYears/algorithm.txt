Problem
A function that determines whether a given year is a leap year or not.


Algorithm
1. Create LEAPYEARS fxn with YEAR parameter.
    - Create if-statement to check if YEAR is a leap year


Pseudo-code
1. 
    FXN leapYears(year)
        IF (year is divisible by 4 AND year is divisible by 400 AND year is not divisible by 100)
            RETURN true;
        ELSE 
            RETURN false;
    END
